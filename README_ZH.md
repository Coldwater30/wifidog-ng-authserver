# wifidog-ng-authserver
一个简单的认证服务器，用于测试wifidog-ng

# 部署
## 根据自己的平台下载编译好的文件

https://github.com/zhaojh329/wifidog-ng-authserver/releases

## 然后解压

    tar zxvf wifidog-ng-authserver-x64.tar.gz

## 运行

    ./wifidog-ng-authserver

## 在你的路由器上配置  wifidog-ng 

	vi /etc/config/wifidog-ng
